

Установка сайта:
	1. Скачай XAMPP https://www.apachefriends.org/ru/download.html
	2. После установки найди папку htdocs, по умолчанию C:\xampp\htdocs
	3. Закинь папку "cloudstorage" которую я скинул в эту папку
	4. Запусти XAMPP и нажми на START в "Apache" и "MySQL"
	5. Загрузи Базу Данных (Снизу написано как)
	6. После этого в браузере перейди по ссылке http://localhost/cloudstorage

Загрузка Базы Данных:
	1. Перейди по ссылке http://localhost/phpmyadmin
	2. Создай Базу данных "cloudstorage" и выбери "utf8_general_ci"
	3. Нажми на ИМПОРТ
	4. В папке есть папка "Database"
	5. Внутри есть файл cloudstorage.sql
	6. Загрузи этот файл и нажми кнопку ВПЕРЕД

Как пользоваться:
	Сперва перейди по ссылке http://localhost/cloudstorage/
	
	Существует аккаунт для тестирования: login: test pass: test
	Регистрация:
		1. Пройди регистрацию
		2. После регистрации тебя перекинет на страницу с логином
		3. Введи свой логин и пароль
		4. После нажатия кнопки тебя перекинет на твою страницу
		5. Проверь в квадрате с данными, выводиться ли твое имя(логин)

	Загрузка фоток:
		1. Нажми на кнопку "Add photo"
		2. Выйдет модуль с загрузкой файла
		3. Загрузи свою фотку
		4. Если выйдет ошибка "Sorry photo is not uploaded!" значит файл слишком большой
		5. Проверь что ты загружаешь png/jpg/jpeg файл
		6. После загрузки твоя фотка должна выведена в таблицу
		7. Эти фотки кроме тебя другие пользователи не видят
		8. Чтобы скачать файл нажми и тебя перекинет на страницу с файлом

	Обратная связь(Feedback):
		1. Обратную связь могут оставлять ВСЕ пользователи
		2. Для доступа в список со всеми сообщениями зайди в Админ панель(login: admin pass: admin)
		3. Можно удалить обратную связь нажав на кнопку Delete

	Сессия:
		1. Если ты уже авторизован и вышел из сайта, то при нажатий на sign in тебя сразу перекинет на твой профиль
		2. Соответственно наоборот, те кто еще не авторизовался никак не могут попасть на профиль через URL
